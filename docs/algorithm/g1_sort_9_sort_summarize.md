# 排序总结&避坑指南



|            | 时间复杂   | 额外空间复杂 | 稳定性     |
| ---------- | ---------- | ------------ | ---------- |
| 选择排序   | O(N^2)     | O(1)         | 无         |
| 冒泡排序   | O(N^2)     | O(1)         | 有         |
| 插入排序   | O(N^2)     | O(1)         | 有         |
| 归并排序   | O(N*logN)  | O(N)         | 有         |
| 随机快排   | O(N*logN)  | O(logN)      | 无         |
| 堆排序     | O(N*logN)  | O(1)         | 无         |
| ========== | ========== | ==========   | ========== |
| 计数排序   | O(N)       | O(M)         | 有         |
| 基数排序   | O(N)       | O(N)         | 有         |
